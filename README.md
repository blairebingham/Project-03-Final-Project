# Project 03 - Final Project
This is the live source code for a fictional business website, hosted on Github.

Live Site: 

## Project Purpose 
A simple two-page website for the fictional business, "The Coffee Nook", built as a starter project to practice HTML structure, internal linking, and Github deployment.
## Features 
- **Homepage (`index.html`)**: Features a welcoming hero section and a list of daily specials.
- **About Page (`about.html`)**: Contains the business origin story and an HTML table for store hours.
- **Navigation**: A consistent menu bar that allows seamless jumping between pages.
## Development Setup 
This setup is used for local development and site maintenance. 
### Prerequisites 
- **VS Code:** For code editing and file management.
- **Github Desktop:** For version control and syncing changes.
- **Live Server (VS extension code):** For live local previews. 
### Installation
- **Clone the Repo:** Open **Github Desktop** and clone this repository to your computer.
- **Open in VS Code:** In Github Desktop, click the **"Open in Visual Studio Code"** button.
- **Run Locally:** Right-click `index.html` and select **"Open with Live Server"** to view changes in your browser.
